# JED Description Draft

TreeK Pro adds a post tree to Kunena topics. This helps users see the structure of responses, switch between related posts, and navigate long discussions without losing the context of who replied to whom.

The settings of the tree window allow the user to see the content of a single post or teasers (beginnings) of all posts on the topic at once, choose a convenient tree view, immediately receive a message about adding a new post to the topic, etc. without leaving the tree window.

All actions for creating themes, adding and changing posts, etc. are carried out in the usual way in Kunena.

## Short Description

TreeK Pro adds a structured post tree to Kunena Forum topics with advanced tools.

## Main Features

- Open a post tree from Kunena topics.
- Display post connections.
- Highlight the latest and active posts.
- Configure tree display in a settings window.
- Show post text teaser and tooltip.
- Show date, time, and post index.
- Export the tree as text, BBCode, or HTML.
- Navigate from the tree back to the Kunena topic.
- Parent-post navigation in the forum view.
- Screenshot teaser.
- Flat list view.
- Comfort symbols for navigation and highlighting.
- Topic monitoring notice when a post is added.
- Draggable settings window.
- Save and restore settings.
- Comfort form header.
- Subject suffixes.
- Collapsed attachments.
- Inline action buttons.
